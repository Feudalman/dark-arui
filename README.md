# Dark-Arui

**Black enough**

Dark-Arui 是一款专为长时间编码设计的暗色主题，采用统一的深黑背景和低对比度设计，营造出沉浸式的开发体验。以绿色和青色为主的语法高亮既柔和又显眼，让代码阅读更加舒适。

![example](example.png)

## 🎨 颜色方案

### 主色调

- **背景色**: `#0a0a0a` - 深邃的黑色
- **文本色**: `#e0e0e0` - 柔和的浅灰色

### 语法高亮

- **字符串**: `#7fb069` - 柔和绿色
- **关键字**: `#5dbcd2` - 青色
- **函数**: `#52b788` - 明亮绿色
- **数字/常量**: `#d4a574` - 柔和黄色
- **类型/类**: `#6db3d4` - 柔和蓝色
- **注释**: `#6b7d7d` - 灰青色，平衡可见度

### 状态颜色

- **错误**: `#d46a6a` - 柔和红色
- **警告**: `#d4a574` - 黄橙色
- **信息**: `#6db3d4` - 蓝色

## 📦 安装

### 从 VS Code Marketplace 安装

1. 打开 VS Code
2. 按 `Cmd+Shift+X` (macOS) 或 `Ctrl+Shift+X` (Windows/Linux) 打开扩展面板
3. 搜索 "Dark-Arui"
4. 点击安装

### 手动安装

1. 下载最新的 `.vsix` 文件
2. 打开 VS Code
3. 按 `Cmd+Shift+P` (macOS) 或 `Ctrl+Shift+P` (Windows/Linux)
4. 输入 "Extensions: Install from VSIX..."
5. 选择下载的 `.vsix` 文件

## 🚀 使用

1. 安装主题后，按 `Cmd+K Cmd+T` (macOS) 或 `Ctrl+K Ctrl+T` (Windows/Linux)
2. 在列表中选择 "Dark-Arui"

或者通过设置：

```json
{
  "workbench.colorTheme": "Dark-Arui"
}
```

## 🎯 设计理念

Dark-Arui 旨在创造一个极致简洁的编码环境：

- **减少视觉噪音** - 通过统一的背景色和低对比度边框，让你的注意力集中在代码本身
- **护眼设计** - 深黑背景配合柔和的高亮色，适合长时间编码
- **语义清晰** - 绿色表示正常元素，红色仅在需要关注时出现
- **无干扰** - 行高亮不使用边框，hover 效果也极其微妙
